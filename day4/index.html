Immediately Invoked Function Expression

  • IIFE is an anonymous function that is created and then immediately invoked
  • Design pattern which produces a lexical scope using JavaScript's function scoping
  • Used for avoiding variable hoisting and protects against polluting the global namespace

Constructor Pattern

function Property (name, city, state) {

  this.name = name;
  this.city = city;
  this.state = state;

  this.displayPropertyInfo = function () {
    return this.name + ', ' + this.city +  ', ' + this.state;
  };
}

var prop = new Property('Pensacola Place', 'Chicago', 'IL');

console.log(prop.name);
console.log(prop.displayPropertyInfo());

Module Pattern

Using Object Literal Notation

var propertyModule = {

  name: 'Pensacola Place',

  details: {
    city: 'Chicago',
    state: 'IL';
  },

  displayPropertyInfo: function () {
    return this.name + ', ' + this.details.city;
  }
};

var propertyModule = (function () {

  /* private members */
  var city = 'Chicago',
      state = 'IL';

  };

  /* public members */
  return {
    name: 'Pensacola Place',
    displayPropertyInfo: function () {
        return name + ', ' + city;
    }
  };

What is a Angular Service?

  • Service is not a 'web service'
  • Its' a component/object that serves your app
  • Angualr built-in services are prefixed with $

Creating Services

We can define our own services by registering the service's name and service factory function, with an Angular module. 

var myModule = angular.module('myModule', []);

myModule.factory('ServiceName', function() {
  var serviceMember;
  
  var serviceMethod = function () {
	console.log('serviceMethod');
  };
  
  return  {
	serviceMember: serviceMember,
	serviceMethod: serviceMethod
  };
});

We can also define our own services by registering the service's name and service service function, with an Angular module. 

var myModule = angular.module('myModule', []);

myModule.service('ServiceName', function() {
 
  this.serviceMember = '';
  
  this.serviceMethod = function () {
	console.log('newServiceMethod');
  };
});

How to use a service

var myModule = angular.module('myModule', []);

  myModule.controller('MyController', function (ServiceName) {
  
  console.log(ServiceName.serviceMember);

  ServiceName.serviceMethod();
});

How service gets instantiate by Angular

	
Angualr factory function code

function factory(name, factoryFn, enforce) {
	return provider(name, {
		$get: enforce !== false ? enforceReturnValue(name, factoryFn) : factoryFn
	});
}


	
Angualr service function code

function service(name, constructor) {
	return factory(name, ['$injector', function($injector) {
	  return $injector.instantiate(constructor);
	}]);
}

Functionally factory and service both are same.
Prefer factory function over service function, dealing with this keyword is messy.
